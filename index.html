This wallet requires MetaMask

What is MetaMask?

MetaMask is an extension for Chrome, Opera, and Firefox that allows the browser to interact with the Ethereum blockchain. MetaMask also has a built in Ethereum wallet - this is the wallet that your tokens must be stored in to use this wallet.

Why do I need MetaMask?

*token* is an ERC-20 token built on the Ethereum blockchain. This means that, in order to interact with the token, your browser must be able to interact with the Ethereum blockchain. The incredible security of the MetaMask wallet also ensures that your tokens will be as safe as possible.

Is MetaMask safe?

MetaMask is extremely trusted and widely used by the Ethereum community. Your private keys are stored on your machine and encrypted with your MetaMask password, ensuring the security of your MetaMask Ethereum wallet. Many of the Ethereum community's most trusted Ethereum web applications use MetaMask for its ease-of-use and security, including MyEtherWallet. If you have any reservations about the security of MetaMask, feel free to review their GitHub. MetaMask is open source!

For more information, check out the MetaMask website.

You must be logged in to MetaMask

You have MetaMask installed, but you're not logged in. You need to be logged in to your MetaMask Ethereum wallet in order to use Locke Pocket. If you're not sure how to use MetaMask, check out this tutorial. For more information, check out the MetaMask website.
